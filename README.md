Camião De Carga
===============

Para controlar o camião você pode clicar em "cima" e "baixo" para accelerar e voltar para trás, 
e também em "esquerda" e "direita" para curvar em abos os lados.E a função especial é de levantar 
a carroceria do camião e para operar isso você clica em "x" e "z" para levantar e baixar a carroceria
